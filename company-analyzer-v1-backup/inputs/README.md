# inputs/ の使い方

分析したい企業ごとにフォルダを作り、資料を入れる。フォルダ名は半角英数（例: `toyota`, `kawasaki-kisen`, `mitsui-bussan`）。

```
inputs/
└── <company>/
    ├── yuho.pdf        # 有価証券報告書（必須に近い）
    ├── chukei.pdf      # 中期経営計画（あると将来性分析が鋭くなる）
    ├── ir/             # 決算説明会資料・統合報告書など（PDF複数可）
    ├── news.md         # 直近ニュースの要点を貼り付け（任意）
    └── peers.md        # 競合の主要数値（任意・あると比較とNo.1判定が鋭くなる）
```

## peers.md の書き方（任意だが効果大）
競合3〜5社の主要数値を表で貼るだけでよい。例:

| 社名 | 売上 | 営業利益 | 営業利益率 | ROE | ROIC | PER | PBR |
|---|---|---|---|---|---|---|---|
| 当社 | … | … | … | … | … | … | … |
| 競合A | … | … | … | … | … | … | … |
| 競合B | … | … | … | … | … | … | … |

## 資料の集め方（外部API不要・手で集める）
- 有報・統合報告書：各社IRサイト / EDINET から手動DL
- 中計・決算説明会：各社IRサイト
- ニュース：気になる記事の要点を `news.md` に貼る

## 推奨の最初の1社
- **川崎汽船**：景気循環・巨額capital・WACC感応度が効き、海運志望にも直結（Damodaran向き）
- **トヨタ**：モビリティ転換期の将来性分析が映える
