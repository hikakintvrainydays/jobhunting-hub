# 03_moat.md — 競争優位とNo.1要素（W3）

> 担当：W3 競争優位／Healy 第2章。No.1主張は順位・シェア・数値で裏づける。出典：`00_facts.md`、`04_findiag.md`（ピア）。

## 1. 5 Forces（業界の構造的魅力度＝中〜やや低、ただしトヨタの位置取りは良い）
- **既存競争：強い（魅力度↓）**。グローバルで過当競争。FY2026/CY2025は関税・EV減損で**Ford △82億ドル、Honda △3,530億円、Nissan △5,331億円、Stellantis △223億ユーロが純損失**、VWも営業利益率2.8%（`04_findiag.md`）。
- **新規参入：従来は高障壁→BEVで低下（魅力度↓）**。資本・ブランド・販売網・認証が参入障壁だったが、BYD（NEV 460万台、EVでテスラ超え）等のBEV専業が垂直統合×低コストで侵食。
- **代替：中**。BEV・MaaS・ライドシェア・公共交通。トヨタはマルチパスウェイで“代替の方向が一つに決まらない”不確実性をヘッジ。
- **買い手交渉力：弱い（個人は分散）／フリート・ディーラーは中**。
- **売り手交渉力：中→上昇**。電池材料・半導体・労務費の上昇。ただしトヨタは系列（デンソー/アイシン/豊田自動織機）と購買規模で相対的に強い。

## 2. 競争優位の源泉と持続性
| 源泉 | 内容 | 持続性（崩れにくさ） |
|---|---|---|
| 規模の経済 | 世界販売台数No.1（グループ約1,123万台）、購買・開発の固定費分散 | 高（ただし中国勢の台頭で相対化） |
| 無形資産（HEV技術） | 四半世紀のハイブリッド量産・特許・サプライチェーン | **非常に高**（補助金・充電網に依存せず黒字で電動化できる唯一格） |
| コスト優位（TPS） | トヨタ生産方式・原価改善の常設化 | 高（模倣困難な現場ケイパビリティ） |
| 系列・垂直統合 | デンソー/アイシン/豊田自動織機、持分法益5,527億円 | 中〜高 |
| 金融の内製（TFS） | 約39兆円の債権で保有局面まで収益化 | 中（資本を食う） |
| ブランド・品質 | フルラインアップ×全地域、A1/A+格付 | 高 |
| 立地分散 | 日本/北米/欧州/アジア/その他に生産・販売を分散 | 高（地政学・関税のヘッジ） |

## 3. 【最重要】No.1要素 — 3候補→1つに絞る

**候補A：世界販売台数No.1（6年連続）**
- 2025暦年 グループ約1,123万台で世界首位（2位VW約898万台、3位ヒョンデ・起亜約727万台、BYD約460万台）。連結ベースでも959.5万台。
- 弱点：規模は中国勢の台頭で“相対化”しやすく、「だから当社」の決め手として弱い。

**候補B：HEVを軸に“電動化を黒字で量産できる”技術・原価力**
- 補助金や充電インフラに依存しないHEVを四半世紀量産。マルチパスウェイで需要の不確実性をヘッジ。
- 強み：固有性が高く模倣困難。ただし単体では「将来のBEVシフトに弱いのでは」という反論を受けやすい。

**候補C：逆風下でも利益を出し続ける“収益のレジリエンス（利益の質）”** ← ★最有力
- **証拠（FY2026/CY2025、`04_findiag.md`）**：関税1.38兆円の直撃を受けてなお**営業利益3.77兆円・営業利益率7.4%・純利益3.85兆円**を確保。同じ年に Ford/Honda/Nissan/Stellantis は**純損失**、VWは営業利益率2.8%、GMは純利益率1.5%。**主要量産メーカーで唯一、二桁に近い利益体力と黒字を維持**。
- レジリエンスの源泉＝候補A（規模）×候補B（HEV・原価力）×全地域分散×内製金融＝**複合的で他社が一要素だけ真似ても再現できない**。
- 財務体力でも裏打ち：自己資本39.9兆円、現金12.7兆円、格付 Moody's **A1**／S&P **A+**（ともに自動車業界最上位級）。

### 確定：No.1要素＝「**業界が軒並み赤字に沈む逆風下でも、補助金に頼らず黒字を出し切る“収益と財務のレジリエンス”**」
- 1〜2文（面接で言う形）：「トヨタの世界一は“台数”だけではありません。**関税1.38兆円を被ってなお営業利益率7.4%・純利益3.85兆円を出し、同じ年にFord・Honda・Nissan・Stellantisが赤字に沈むなか主要量産メーカーで黒字を守り切った“利益のレジリエンス”が本当の一番**です。その源泉はHEVの量産技術・TPSの原価力・全地域分散・自前の金融という、一つ真似ても再現できない複合優位にあります。」
- なぜこれが“なぜ当社か”の核になるか：規模No.1（候補A）は中国勢で相対化されるが、**「逆風でも利益を出す力」は競合が直近の決算で“出せなかった”ことが証明済み**＝差別化が事実で裏づく。
