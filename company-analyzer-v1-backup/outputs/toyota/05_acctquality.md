# 05_acctquality.md — 会計品質（W5）

> 担当：W5 会計品質／Healy 第3章。不正は断定しない。「数字の質」と「要確認」を冷静に。出典：`00_facts.md`（決算要旨）。

## 総合判定（結論先）
**トヨタの利益数字は概ねそのまま信じてよい質**。IFRS任意適用、会計方針変更・見積り変更・連結範囲の重要変更いずれも「無」（決算要旨 注記事項）。利益はキャッシュを伴っている（OCF/純利益1.37倍）。ただし**FY2026は「一過性の追い風（為替差益・金利スワップ評価益・税の反動）」が利益とCFを下支え**しており、**“素の実力”はヘッドラインよりやや低い**点に注意。

## 1. 保守的/積極的の傾向（業界標準比）
- **総じて保守的寄り**。販管費は減少（4.78兆→4.70兆）させつつ、原価は実需どおり計上。引当・品質保証負債（2.10兆円、前期1.97兆）を厚めに持つ（過去のリコール経験を反映した堅実な姿勢）。
- 減価償却：賃貸資産（リース車両9.7兆円）を含む有形固定資産の償却を着実計上（D&A 2.39兆円）。**capex>D&A（1.73倍）で資産は増加局面**だが、過大な資産計上で費用を繰延べる兆候は見えない。
- のれん：B/Sの無形資産1.39兆円と小さく、**のれん依存・大型減損リスクは低い**（買収でののれん膨張型ではなく自前開発型）。

## 2. アクルーアル（利益と営業CFの乖離）
- FY2026：純利益3.99兆 < 営業CF5.47兆 → **アクルーアルはマイナス（CFが利益を上回る）＝利益の質は良い方向**。
- ただし乖離拡大の主因は **法人税支払の前年反動（△2.50兆→△1.24兆）** と **その他流動負債+1.48兆**＝**タイミング/一過性**。本質的な稼ぐ力の改善ではない。金融債権の増加は△2.02兆のCF流出（構造要因）。

## 3. 一過性要因（利益の嵩上げ/圧迫）
**嵩上げ（＋）方向**：
- **為替差損益（純額）+4,008億円**（前期+7,053億）。水準は縮小したが依然プラスで税引前利益を底上げ。
- **金融セグメント増益の主因＝米販売金融子会社の金利スワップ評価益**（決算要旨 本文明記）＝**評価性・非経常色が濃い**。金融営業利益+1,682億の相当部分がこれ。
- **その他の包括利益+1.53兆円**（在外営業活動体の為替換算差額+9,463億、FVOCI金融資産の評価益）→当期包括利益5.52兆円（+36.4%）は**純利益より見栄えが良いが、多くは換算・評価差額**で現金化されたものではない。

**圧迫（−）方向**：
- **米国関税1.38兆円**（営業利益の減益要因として明記）＝外生的・継続見込み（FY2027見通しに織り込み）。
- 諸経費の増減・低減努力 △2兆300億円（人・未来への投資の拡大）。

## 4. 注記の要確認ポイント
- **「売却目的で保有する資産」2.02兆円が新規計上**（日野の連結除外＋豊田自動織機株）。これは**翌期（FY2027）に売却損益・連結除外影響が出る**ため、FY2027のP/L・B/Sは**非連続（一過性の売却損益や持分の変動）**になる見込み＝**翌期決算は“素の実力”との切り分けが必要**。
- 持分法投資損益5,527億円（営業利益の約15%）。デンソー/アイシン等の業績次第で振れる“外部依存の利益”である点は意識。

## 5. 要確認（資料制約）
- 政策保有株式の銘柄数・時価・含み損益の明細は**有価証券報告書「株式の保有状況」で要確認**（決算要旨・半期報告書には個別開示なし）。
- セグメント別の正確な資本配賦・税負担。自動車/金融の区分P/Lはあるが、内部金利・配賦前提は非開示＝W4のROICは事業ICベースの近似。
