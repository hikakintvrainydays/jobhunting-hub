## アカウント設計

### 就活リサーチ専用アカウントを作る理由
- 自分の本アカウントに就活情報が混じると、フォロワーに就活状況がバレる
- 採用担当者が本アカウントを見たとき、就活臭い投稿・愚痴がない状態を保つ
- 匿名で先輩/OBにDMできる

### アカウント設定
| 項目 | 設定値 |
|------|--------|
| ハンドル名 | `shukatsu28` 系の匿名名（実名禁止） |
| 表示名 | 「28卒就活生」等の一般的な名前 |
| 登録メール | 就活専用Gmail（例: `shukatsu.matui28@gmail.com`） |
| 電話番号 | 自分のサブSIM or 一時番号サービス |
| プロフィール | 「商社・コンサル志望 / 28卒 / 滋賀大」程度（大学名は任意） |
| 公開設定 | **公開**（採用公式をフォローするため。鍵アカは情報収集に不便） |
| アバター | アニメ・抽象画 等（顔写真禁止） |

## 情報収集ルーティン（週次）

毎週月曜に実施。所要30分。

```
1. ターゲット企業採用公式アカウントの直近投稿を確認（5〜10社）
2. 「28卒 インターン 締切」で検索 → 新着情報をtargets.mdに追記
3. 「企業名 筆記 テスト 28卒」で各社のテスト形式最新情報を確認
4. OB訪問が来ていたらDM返信
5. 気になる投稿は「ブックマーク」で保存（スクリーンショット > URL保存）
```

## DM/OB接触テンプレート

### 初回DM（就活先輩へ）
```
はじめまして、28卒で[業界/企業]を志望している[匿名ハンドル]と申します。
[企業名]の選考フロー・筆記試験の形式について
実際に選考を経験された方のお話を聞かせていただけますか？
30分程度オンラインでお時間いただけますと幸いです。
よろしくお願いいたします。
```

### 注意
- 一度に大量のDMを送らない（スパム判定・採用担当にバレるリスク）
- 1日3通まで
- 断られたら即引く
