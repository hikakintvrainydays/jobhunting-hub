## AI面接 注意点チェックリスト

### 技術・環境（録画前）
- [ ] Wi-Fi有線推奨・充電満タン
- [ ] 照明: 顔に光が当たる設定（逆光NG・白壁前が最安定）
- [ ] 背景: 無地・白（散らかった部屋はマイナス）
- [ ] カメラ目線: **レンズを見る**（画面の自分を見ると下向きに見える）
- [ ] 服装: スーツ（上半身のみでも）
- [ ] テスト録画1本で画角・音量・照明を確認してから本番

### 話し方・表情
- [ ] 笑顔は普段の1.2倍（対面より感情が伝わりにくい）
- [ ] 抑揚をつける・語尾まで落とさない
- [ ] 冒頭3秒で「こんにちは、松井大と申します」
- [ ] 60秒なら45秒で本題を終え15秒で締める
- [ ] 撮り直しは迷わず使う（締切当日は使わない）

### STAR構造（各回答の基本）
```
S 状況  （5秒）: 何をやっていたか
T 課題  （5秒）: 何が問題だったか
A 行動 （30秒）: 自分が何をしたか ← ここが核心
R 結果 （10秒）: 数字で語る（50万円採択など）
締め   （10秒）: 住商でどう活かすか
```

### 住商固有の注意
- [ ] KINTO50万円採択という**具体的な数字を冒頭15秒以内**に出す
- [ ] 「住商でやりたいこと」は動画では問われない可能性高い → 自己PRに集中
- [ ] 一次予選はGD重視 → 動画でアピールした「個性」をGDで「チームへの貢献」に転換する準備
