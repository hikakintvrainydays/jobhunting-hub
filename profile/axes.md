# 就活の軸と志望理由の核

## 軸（表向き・面接用）
1. **現場の課題を、複数の利害関係者を巻き込んで事業として解決したい**（ToruTour原体験: 観光客×店舗×言語の壁を接続した）
2. **海外と日本の接続で価値を出したい**（海外で助けられた原体験 → 日本でインバウンド支援 → 海外現場での検証へ）
3. **財務・データの専門性で意思決定に踏み込みたい**（宮西ゼミ×自作の企業分析基盤）

## 本音（戦略用・外に出さない）
- 強い場所にいたい／人生の選択肢を最大化したい／承認・優越 → 面接語彙では「大きな責任を背負える環境で事業と人を動かす力を付けたい」に翻訳

## 業界接続
- **第一軸=総合商社**: 軸1×2×3が全部刺さる。トレード+事業投資の「現場で仕組みを作って稼ぐ」側
- **第二軸=海外事業に接続するJTC**: 海運(郵船/商船三井/川崎汽船)・トヨタ・食品(サントリー等)・エネルギー
- **第三軸=AI/DXを事業実装する企業**: NTTデータ・AWS等
- 「なぜ起業でなく就職?」→ 個人の限界(資本・信用・スケール)を現場で痛感 → 商社の機能(信用・金融・ネットワーク・人材)で同じ構造の課題解決を桁違いの規模でやりたい
