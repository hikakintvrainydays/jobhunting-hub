# 松井大 自己インタビューシート
# — 想定問答の素材を掘り出すための一日作業 —

> **使い方**
> - 全部埋める必要はない。思い出せるところから書く。
> - 文章じゃなくていい。箇条書き・キーワードでOK。
> - 「わからない」「覚えてない」も書いていい。
> - 埋めたら `/qa fill 志望理由` などで回答生成に使う。
>
> **一番大事なフレーム（各エピソード必須）**
> ```
> 違和感  → 「なんかおかしい」「なんでこうなってるんだ」と思った瞬間
> 仮説    → 「たぶん原因はこれだ」「こうすれば変わるはず」と考えたこと
> 調整    → 実際に試したこと、失敗して変えたこと、捨てた選択肢
> 実行    → 最終的にどう動いたか（具体的な行動・セリフ・数字）
> ```
