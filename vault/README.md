# 企業分析 Obsidian 保管庫

**このフォルダをObsidianで開く**: Obsidian → 「別のVaultを開く」→ このフォルダを選択

## 構成

```
vault/
├── companies/          ← 企業ごとの面接台本（company-analyzerが自動生成）
│   ├── トヨタ.md
│   ├── 三菱商事.md
│   └── ...
└── README.md
```

## 使い方

1. `company-analyzer` で「分析: <企業名>」を実行
2. → `vault/companies/<企業名>.md` が自動生成される
3. → `company-analyzer/outputs/<企業名>_財務分析.xlsx` も同時生成される
4. Obsidianで台本を読み込み → フラッシュカードセクションを暗記
5. 面接前に宮西先生とZoom → Excelを見せて話し方を相談

## タグ体系

- `#企業分析` — 全企業ノートに付与
- `#28卒` — 今年度向け
- `#商社` / `#メーカー` / `#金融` — 業種タグ

## 財務指標の見方（共通）

| 指標 | 安全ライン | 危険ライン |
|------|----------|---------|
| ICR | 3倍以上 | 1倍以下 |
| 自己資本比率 | 40%以上 | 10%以下 |
| PBR | 1倍以上 | 0.5倍以下（解散価値割れ） |
| ROE | 8%以上（WACC目安） | 5%以下 |
| ROIC > WACC | 価値創造 | ROIC < WACC = 価値毀損 |
