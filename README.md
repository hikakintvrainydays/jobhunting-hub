# JOB-HUNTING-HUB
松井大の就活を一元運用する Claude Code プロジェクト。**「企業を知る」= company-analyzer、「自分を売る」= 本ハブ**、の分業で28卒・総合商社内定を取りに行く。

## これは何か（30秒）
- `profile/facts.md` が**唯一の真実（SSOT）**。全実績に ✅確定/⚠️要確認/💭構想/⏸停止 の確度タグ。ESに使えるのは✅だけ。
- `/es` でES生成→鬼添削→証拠照合まで自動。`/drill` で商社面接官との模擬面接。`/today` で毎朝5分の作戦会議。
- 既存の **company-analyzer** は `company-analyzer/` に置けば `/analyze` から接続される。

## セットアップ（今夜やる3つ）
1. このフォルダを Claude Code で開く（`claude` を実行）。
2. 既存の company-analyzer 一式を `company-analyzer/` に移す（任意だが推奨）。
3. 下の**初回プロンプト**を貼る。⚠️項目の質問に答えていけば facts.md が確定していく。

### 初回プロンプト（コピペ用）
```
JOB-HUNTING-HUBを初期化して。
1. README.md と REQUIREMENTS.md を読んで全体像を把握
2. profile/facts.md の ⚠️要確認 項目を1つずつ俺に質問して確定値に更新（TOEIC最新・帰国日・タイツアー採否・大和との会話状況など）
3. pipeline/companies.csv の締切を今日の日付基準で精査し「要調査」の埋め方を提案
4. 最後に /today を実行して今日のアクションを出して
```

## 毎日の運用
| いつ | コマンド | 何が起きる |
|---|---|---|
| 毎朝5分 | `/today` | 締切スキャン→今日の3タスク |
| ES書く時 | `/es 三菱商事 学生時代に力を入れたこと 400` | 生成→採点→修正版→証拠照合→保存 |
| 面接前 | `/analyze 三井物産` → `/drill 三井物産 1次` | 企業分析→深掘り模擬面接 |
| 実績が増えた | `/fact` | 証拠つきでSSOTに登録 |
| 日曜15分 | `/weekly` | 進捗棚卸し→来週の時間配分 |

## 思想（なぜこの形か）
1. **誇張は構造で防ぐ**: 825/935混在・事業の停止/継続の揺れは、本人の記憶でなく台帳で管理する。fact-checker が❌を1つでも見つけたら提出不可。
2. **拡散は枠で防ぐ**: S5社/A8社/B7社の20社上限。増やすなら落とす。
3. **蓄積で勝つ**: ES・問答・講評は全部ファイルに残る。3ヶ月後の自分が今日の自分より強い状態を作る。
4. **先輩20名がベンチマーク**: `benchmarks/` の26卒プロフィールと自分を常に比較し、「国際派テンプレ」と被らない見せ方を選ぶ。
