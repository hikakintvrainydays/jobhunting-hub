この表は、三菱商事の採用パターンを分類したものです。一般的に知られています。

| パターン     | 特徴                  | 代表例                    |
| -------- | ------------------- | ---------------------- |
| エリート型    | 東大・京大・一橋等の最難関国立大学出身 | 東大法学部、京大経済学部、一橋商学部     |
| 体育会リーダー型 | 早慶等の有名私立大学で体育会の主将経験 | 慶應ラグビー部主将、早稲田野球部主将     |
| 国際派型     | 海外留学経験や高い語学力を持つ     | TOEIC950点、交換留学1年、多言語対応 |
| 起業家型     | 学生時代から起業経験や事業立ち上げ   | 学生起業、NPO設立、事業コンテスト優勝   |
| 専門性型     | 特定分野で高い専門性を持つ       | 公認会計士、ITスキル、金融知識       |
|          |                     |                        |
**宮西ゼミの過去の商事採用は、国際派型＞専門性型＞企業家型、です。営業（50％）、人事、財務経理（30％）、リスクマネジメント、などに配属になっています。財務経理の配属は、東大(８０％)・一橋に限定されていて、それ以外の大学卒は稀です。商社に就職した先輩の30%は財務経理配属です。例えば、丸紅や住友商事の場合、８割の先輩が財務経理配属です。伊藤忠商事で５０％財務経理です。

商事、物産、伊藤忠は、配属先で基本的にはずっと働きます。丸紅と住商の場合、入社後３経過後に希望する部署を選ぶことができるので、他部門に移動できます。また、伊藤忠商事は、入社後どこかのタイミングで１－２年程度は経理部で働きます。経理の知識を全員に入れるためで、７年前からこの制度が始まりました。**
