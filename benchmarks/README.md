# 26卒先輩ベンチマーク（宮西ゼミ20名）
`26卒_seniors.csv` = ゼミ26卒の就活プロフィール（出身高校／中高大の実績／内々定先／留学先／インターン）。

## 使い方
1. **テンプレ検知**: es-reviewer はこのデータと照合し、「フィリピン留学＋新興国インターン＋体育会役職＋TOEIC900」型と被る表現を減点する。同じ土俵で量(期間・役職)勝負をしない。
2. **差別化の根拠**: 松井の勝ち筋は KINTO受賞(外部評価✅)×Vibe coding(自作✅)×財務専門性 の「専門性×起業家型」。先輩に同型がいないことを確認して使う。
3. **到達水準の参照**: 内々定先(三井物産・丸紅・商船三井・トヨタ・ENEOS等)と実績の組み合わせから、各社の「足切りライン」を逆算する。
