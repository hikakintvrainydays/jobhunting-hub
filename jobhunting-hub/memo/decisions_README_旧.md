# 意思決定ログ

重要な判断を日付ファイルで残す。  
ファイル名: `YYYYMMDD_<テーマ>.md`

## テンプレート

```markdown
---
date: YYYY-MM-DD
topic: <テーマ>
decision: <決定内容1行>
tags: [decision]
---

## 背景
（何を判断しなければならなかったか）

## 選択肢
- A: ...
- B: ...

## 決定
**→ A** を選んだ。理由: ...

## 再検討トリガー
（どうなったら見直すか）
```
