## 4. AIケース面接 対策

### 評価軸（5つ）
1. **幅出し力** — 論点を何個列挙できるか
2. **構造化力** — MECEに整理・優先順位付け
3. **数的妥当性** — フェルミ推定レベルの根拠
4. **国理解** — 対象国の経済・産業構造
5. **判断の一貫性** — AIが揺さぶっても基準がブレない

### 典型ステップ
```
Step1: 投資先の国・産業が提示される
Step2: 「なぜこの国に投資すべきか」の論点出し（幅出し）
Step3: AIが深掘り「市場成長の要因は？」「競合との差別化は？」
Step4: AIが逆張り「インフレ急騰したら？」「規制が変わったら？」
Step5: 最終的に投資すべきかYes/Noと理由
```

### 27卒の実例（⚠️X情報・確度中）
**テーマ**: サブスク/カーシェアで流通量低下が懸念される北米中古車販売における金融事業の新規事業提案

**選択肢**:
- ① 中古車販売に追加サービス
- ② 現地トレンドを踏まえた新セグメントへの進出
- ③ 中古車販売で新たなパートナーとの連携

**AIが聞いてきた質問**:
- 知りたい前提情報は？
- どのように情報を収集するか
- 得た情報をどう分析するか
- 中古車ローン事業の周辺領域に進出するとすればどんな事業か
- どのセグメントか / どんなパートナーと提携すべきか
- 新規事業で注意すべき項目は？
- **現地でデモにより営業所が封鎖、現地メンバーと交流が途絶えた場合どう対応するか**（危機対応）
- 本社にはどのような情報を提供するか

- **在り姿**: 「商社の枠組みを超える価値創造企業グループへ」

### Global crossvalue platformとは
「丸紅グループを一つのプラットフォームとして捉え、グループの強み、社内外の知、ひとり一人の夢と夢、志と志、さまざまなものを**縦横無尽にクロスさせて新たな価値を創造**する」
